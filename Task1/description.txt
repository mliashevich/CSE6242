NodeEdgesWeightMapper - отвечает за map.
На вход получаем строку из файла - value.
Из которой достаем нужные нам два значения:
nodeId - вершина
weight - вес ребра

NodeEdgesWeightReducer - отвечает за reduce.
на вход получаем key - id вершины и values - список весов ребер, которые были найдейны для этой вершины.
Суммируем веса и на выход отдаем пару: (id вершины, сумма весов ребер)
